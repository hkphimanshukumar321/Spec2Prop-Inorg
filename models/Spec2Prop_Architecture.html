Figure: Architecture of the Spec2Prop-Edge screening pipeline. Raw Raman/XRD spectral files are parsed and preprocessed at runtime into fixed 2048-point spectral vectors. PCA, peak-based, band-ratio, moment, and compact spectral features are concatenated into a 158-dimensional representation and classified using a LightGBM-DART model. The system returns top-k inorganic family suggestions with confidence scores and a screening-level recommendation for scientist verification. The bottom panel illustrates the dataset-backed validation where models trained on the Spec2Prop-InorgBench are deployed on a web UI for held-out test samples.
